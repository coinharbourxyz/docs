Callback Functions

__init__(self, trading_interface: TradingInterface)

Initializes the algorithm with the trading interface.

on_ticker_recd(self, ticker_data)

Called when a new ticker data is received. The ticker_data object contains fields as described in the Dataset Fields section.

on_algo_start(self)

Called when the algorithm starts.

on_algo_stop(self)

Called when the algorithm stops.

on_order_success(self)

Called when an order is successfully executed.

on_order_error(self, error)

Called when an error occurs during order execution.

Dataset Fields

The ticker_data object contains the following fields:

Field Type Description
open float Opening price
close float Closing price
high float Highest price
low float Lowest price
volume float Trading volume
date_time string in ISO 8601 format Time of the data point

Helper Methods

buy(orderType: str, price: float = None)

Buys ETH with all available USDC.

Input Parameters:

  • orderType (string): The type of order to buy ('market' or 'limit').
  • price (float, optional): The price to buy at (only used for limit orders).

Output:

void

self.trading_interface.buy(order_type="market")

sell(orderType: str, price: float = None)

Sells all available ETH for USDC.

Input Parameters:

  • orderType (string): The type of order to sell ('market' or 'limit').
  • price (float, optional): The price to sell at (only used for limit orders).

Output:

void

self.trading_interface.sell(order_type="limit", price=ticker_data["close"])

get_current_balance(asset: str)

Retrieves the current balance of a specified asset.

Input Parameters:

  • asset (string): The asset to get the balance of (either 'usdc' or 'eth').

Output:

float

balance = self.trading_interface.helper.get_current_balance("eth")

Pricing

Transaction fees for each trade is approximately 0.0015USD (INR 0.13)
